# CR-CC — Compte Rendu Conseil de Classe

Application web gratuite pour les parents délégués.

## Installation rapide

1. **Fork** ce repo GitHub sur votre compte
2. Activez **GitHub Pages** (Settings → Pages → main branch)
3. Créez un projet **Firebase** sur [firebase.google.com](https://firebase.google.com)
   - Activez Firestore Database (mode test)
   - Créez une application Web et copiez les clés
4. Ouvrez `votre-site/admin.html`
5. Entrez vos clés Firebase → l'app est prête !
6. Configurez votre établissement dans le panneau admin

## Première connexion

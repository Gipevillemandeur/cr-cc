📘 Guide d'utilisation

💾 Sauvegarde automatique : Vos données sont sauvegardées automatiquement à chaque modification. Si vous fermez l'onglet par accident, ne vous inquiétez pas — à la prochaine ouverture du site, il vous proposera de reprendre où vous en étiez.

1. Écran d'accueil : Sélectionnez votre classe dans la liste, choisissez le trimestre et la date du conseil, puis entrez le code d'accès transmis par le GIPE. Cliquez sur "Commencer" — la liste des matières et des professeurs se charge automatiquement.

2. Saisie (partie JAUNE ✍️) : Remplissez le formulaire :
• Équipe pédagogique : par défaut tous les professeurs sont présents (✓ vert). Cliquez sur le bouton pour passer en absent (✗ rouge).
• Participants : saisissez les noms des parents et élèves délégués, un par ligne.
• Synthèse : indiquez le nombre de félicitations, compliments, encouragements et avertissements.
• Observations : remplissez les blocs pour la direction, le professeur principal, les élèves et les parents.
Tout ce que vous saisissez apparaît automatiquement dans l'aperçu gris à droite.

3. Listing élèves : Cliquez sur ce bouton pour ouvrir une feuille de suivi par élève, imprimable, avec des cases à cocher (Félicitations, Compliments, etc.).

4. Relecture avec le 2ème parent (optionnel) 🔗 : Cliquez sur "📨 Envoyer au GIPE & télécharger" puis sur "Aperçu". Dans la fenêtre qui s'ouvre, vous verrez un lien de relecture. Copiez-le et envoyez-le au 2ème parent délégué par SMS ou email.
Le 2ème parent ouvre le lien, consulte le compte rendu, peut le modifier si besoin, puis vous renvoie un lien.
Quand vous recevez ce lien retour, les modifications apparaissent en rouge barré et en vert pour que vous puissiez les repérer facilement.

5. Envoi au GIPE 🚀 : Quand le compte rendu est prêt, cliquez sur "📨 Envoyer au GIPE & télécharger".
• Si le 2ème parent n'a pas encore relu → une fenêtre vous demande si vous souhaitez d'abord lui envoyer pour relecture.
• Cliquez sur "Non, j'envoie au GIPE" → le PDF est téléchargé sur votre appareil.

Sur ordinateur 💻 : le PDF est aussi envoyé automatiquement à . Vous n'avez rien d'autre à faire !

Sur téléphone 📱 : le PDF est téléchargé sur votre appareil. Une fenêtre s'ouvre pour vous proposer d'ouvrir votre messagerie avec l'adresse du GIPE déjà pré-remplie. Il vous suffira de joindre le PDF téléchargé et d'envoyer !
